# Bookery - Online Bookstore

> A responsive e-commerce web application for browsing and purchasing books, built with HTML, CSS, Tailwind CSS, and JavaScript.
---

## About the Project

**Bookery** is a student web design project that simulates a modern online bookstore. Users can browse book listings, view book details, register or log in to an account, and navigate a clean, responsive storefront, all with Dark/Light mode support.

This project was developed as a group assignment for the **Web Design** subject at **ISTAD (Institute of Science and Technology Advanced Development)**, Year 1, Semester 1.
